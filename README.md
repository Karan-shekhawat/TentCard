# Tent Card Generator

Create professional printable name plates / tent cards in seconds. Adjust size according to your actual name plate holder and print directly from your browser.

## Features

- Add names manually or in bulk
- Customize plate dimensions, colors, and fonts
- Preview with rulers for accurate sizing
- Print-ready output

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm run dev
   ```
